# AI副業・フリーランス地雷地図 v0.1

広告文や案件文を貼って、初心者が「仕事募集なのか」「講座・高額サポート入口なのか」「情報不足なのか」を分解する静的Webアプリです。

## ファイル

- `index.html`
- `style.css`
- `app.js`

## 使い方

1. 3ファイルを同じフォルダに置く
2. `index.html` をブラウザで開く
3. GitHub Pagesに置けばスマホでも使える

## v0.1の考え方

- AI APIは使わない
- 外部通信なし
- ルールベース判定
- まずはプロトタイプとして「赤札」を見る
